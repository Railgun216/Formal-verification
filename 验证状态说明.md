## 如果遇到问题

### 问题 1: ImportError
```
ModuleNotFoundError: No module named 'numpy'
```
**解决**: 运行 `pip install numpy scipy`

### 问题 2: 没有输出
如果运行后没有任何输出，可能是：
- PowerShell 输出缓冲问题（正常，代码仍在运行）
- 检查是否有错误信息

### 问题 3: 验证结果总是 UNKNOWN
这是正常的，因为：
- 过度近似可能不够精确
- 可以启用 `use_exact_check=True` 和 `use_cegar=True` 提高精度
